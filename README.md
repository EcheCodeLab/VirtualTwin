# VirtualTwin

Proyecto base para un **Virtual Twin** orientado a visualización y prueba de interfaces. Actualmente incluye una maqueta HTML con soporte de *popups*, parámetros y ajustes de escalado.

## Contenido

- `UIpruebav10_popups_params_hw_console_scaling.html`: prototipo principal de la interfaz.

## Requisitos

- Navegador moderno (Chrome, Edge, Firefox, Safari).

## Uso rápido

1. Abrí el archivo `UIpruebav10_popups_params_hw_console_scaling.html` en tu navegador.
2. Interactuá con los controles disponibles en la interfaz para validar comportamiento y visualización.

## Objetivos del repositorio

- Servir como base para iterar el diseño y comportamiento de un Virtual Twin.
- Mantener una única fuente de verdad para el prototipo visual.
